📷 Step 1 — Input Capture with OpenCV

How it works

  • OpenCV (cv2) opens a VideoCapture stream from webcam index 0 or a video file path
  • Each frame is read as a BGR NumPy array and converted to RGB for MediaPipe
  • FPS is calculated using time.time() between frames and overlaid on screen
  • Supports static images via cv2.imread() too — same pipeline applies

Code Snippet

import cv2, time

cap = cv2.VideoCapture(0)
prev_time = 0

while cap.isOpened():
  ret, frame = cap.read()
  rgb = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)

  # FPS calculation
  curr = time.time()
  fps = 1 / (curr - prev_time)
  prev_time = curr
🦴 Step 2 — Landmark Detection via MediaPipe

MediaPipe Pose

  • Google's MediaPipe Pose model detects 33 body landmarks per frame
  • Each landmark has normalised (x, y) coordinates plus depth (z) and visibility score
  • The model runs BlazePose — a lightweight neural net optimised for mobile/real-time
  • Key landmarks used: shoulders (11,12), elbows (13,14), wrists (15,16), hips (23,24), knees (25,26), ankles (27,28)

Code Snippet

import mediapipe as mp

mp_pose = mp.solutions.pose
pose = mp_pose.Pose(
  min_detection_confidence=0.5,
  min_tracking_confidence=0.5
)

results = pose.process(rgb_frame)
landmarks = results.pose_landmarks.landmark

# Get shoulder position
shoulder = landmarks[mp_pose.PoseLandmark
    .LEFT_SHOULDER.value]
📐 Step 3 — Angle Calculation

The Math

  • Three landmarks define each joint angle: point A (proximal), B (joint), C (distal)
  • Two vectors are formed: BA and BC from landmark coordinates
  • The angle between them is: θ = arctan2(C-B) − arctan2(A-B), converted to degrees
  • NumPy's np.degrees() and np.arctan2() are used for precision
  • Result is kept in 0–360° range by taking absolute value

Angle Function

import numpy as np

def calculate_angle(a, b, c):
  a = np.array(a) # shoulder
  b = np.array(b) # elbow (joint)
  c = np.array(c) # wrist

  radians = np.arctan2(
    c[1] - b[1], c[0] - b[0]) -
    np.arctan2(
    a[1] - b[1], a[0] - b[0])

  angle = np.abs(np.degrees(radians))
  if angle > 180: angle = 360 - angle
  return angle
✅ Step 4 — Pose Classification

Classification Logic

  • Calculated angles are compared against pre-defined thresholds for each pose
  • All conditions for a pose must pass simultaneously (AND logic)
  • The matched pose name + confidence is overlaid using cv2.putText()
  • A coloured bounding indicator changes per pose for clear visual feedback
  • If no pose matches, "Unknown Pose" is displayed with a neutral indicator

Code Snippet

def detect_pose(angles):
  lk, rk = angles['left_knee'], angles['right_knee']
  ls, rs = angles['left_shoulder'], angles['right_shoulder']
  le, re = angles['left_elbow'], angles['right_elbow']

  # Warrior II check
  if (165 <= le <= 195 and 165 <= re <= 195
    and 80 <= ls <= 110
    and (90 <= lk <= 120 or 90 <= rk <= 120)):
    return "Warrior II"

  # T-Pose check
  if (165 <= le <= 195 and 80 <= ls <= 110
    and 160 <= lk <= 195):
    return "T-Pose"

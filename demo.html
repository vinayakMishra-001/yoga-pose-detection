📡 ESP32-CAM Setup

1. Set ESP32_IP in app.py to your board's IP.

2. Flask polls the MJPEG stream, annotates with MediaPipe skeleton overlays, exposes at /latest_frame.

3. Browser polls /latest_frame every 150 ms.

Endpoints used:
GET /latest_frame → annotated JPEG
GET /pose_data → JSON angles & pose

Run Locally

Install deps, start Flask, then open this page. For ESP32-CAM, set ESP32_IP in app.py before running.

# Install dependencies
pip install flask flask-cors mediapipe opencv-python numpy

# Set ESP32 IP in app.py (ESP32-CAM mode only)
ESP32_IP = "http://<your-esp32-ip>"

# Start Flask server
python app.py

# Open in browser
http://127.0.0.1:5000
